# Markout Generator

The Markout Generator creates scaled stage plots showing rigging point positions. Import your data from CSV and export professional markout sheets for load-in.

## Getting Started

1. Prepare a CSV file with your rigging point data
2. Click **Import CSV** to load your points
3. Review the generated markout plot
4. Export to PDF for your crew

## CSV Format

Your CSV should include columns for:
- **Label** — Point identifier (e.g., "M1", "E2")
- **X** — Position from center line (positive = stage right)
- **Y** — Position from downstage edge (positive = upstage)
- **Type** — Point type (motor, stinger, dead hang)
- **Load** — Weight at this point
- **Trim** — Target trim height
- **Notes** — Optional notes

Click **Download Template** to get a pre-formatted CSV template.

## The Stage Plot

The markout displays:
- Stage orientation (upstage/downstage, stage left/stage right)
- Center line reference
- Each rigging point plotted to scale
- Point labels, types, and loads
- Color-coded legend by point type

## Paper Sizes & Units

Select your preferred paper size and measurement units:
- **Paper**: Letter, A4, or Tabloid
- **Units**: Feet or Meters

## Push to Pull Sheet

Click **Push to Pull Sheet** to send all markout points directly to the Pull Sheet. This creates a complete hardware list from your markout data without re-entering information.

## Vectorworks Integration

Click **Create a hoist report in Vectorworks** for instructions on exporting your data in a format compatible with Vectorworks CAD software.

## Export Options

### Markout PDF
The standard markout plot showing all rigging points on a stage diagram.

### Load-In PDF
A load-in specific version optimized for on-site reference during rigging setup.

## Tips

- Verify CSV data before importing — check for typos in coordinates
- Use consistent labeling conventions across your crew
- Print markouts at full scale when possible for on-site reference
- Keep a digital copy accessible on-site for quick reference
- Coordinate markout positions with the venue's available rigging points
