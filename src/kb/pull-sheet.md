# Pull Sheet Generator

The Pull Sheet generates a complete rigging hardware list based on your point loads. It calculates the exact quantity of shackles, slings, hoists, steel, and other gear needed for your rig.

## Getting Started

1. Enter your rigging points with loads (or push from Point Load Estimator)
2. Configure hoist and hardware preferences
3. Review the auto-generated pull sheet
4. Export to PDF for your crew

## Input Fields

For each rigging point, you can specify:
- **Label** — Name or ID for the point (e.g., "DS Truss P1")
- **Type** — Motor, stinger, dead hang, etc.
- **Load** — Weight at this point (lbs or kg)
- **Trim** — Target trim height
- **Notes** — Any special instructions

## Auto-Calculations

The pull sheet automatically calculates hardware quantities based on your points:
- **Shackles** — 10 per rigging point
- **Steel slings** — 5 per rigging point
- **Hoists/Motors** — 1 per motor point
- **Spansets** — Based on trim height and configuration
- **Baskets** — For beam attachments

## Push from Point Load

After running the Point Load Estimator, click **Push to Pull Sheet** to automatically transfer all calculated points into the Pull Sheet. This saves time and prevents manual entry errors.

## Exporting

Click **Export PDF** to generate a professional pull sheet document. The PDF includes:
- Project header with your details
- Complete hardware list with quantities
- Individual point details
- Notes and special instructions

## Tips

- Double-check hardware quantities against your actual inventory
- Add extra shackles and slings as spares (industry standard is 10-15% extra)
- Include rigger notes for any non-standard configurations
- Review the pull sheet with your head rigger before load-in
